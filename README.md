# Games

Compete with other participants by battling bots. A list of available games is provided below.

### [Gravitrips](./gravitrips/)
Place falling pieces on a 7 wide 6 high board. The first player to place 4 spatially consecutive pieces in a row, column, or diagonal wins.

## Build a bot
To create a bot acquire a wit file for the game, and read the implementation instructions in the package for that game. Example bots will be provided in rust under the examples directory. Bots are submitted as compiled wasm component-model module binaries, so there are some language options. Supported languages can be found in the component-model [documentation](https://component-model.bytecodealliance.org/language-support.html). Be sure to test your bot thoroughly. Onces a bot is ready for submission follow the submission steps below.
