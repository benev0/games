# Games

Games to be played by bots. To create a bot acquire a wit file for the game. Read the instructions in the package. Example bots will be provided in rust. Language support can be found in the component-model [documentation](https://component-model.bytecodealliance.org/language-support.html).

Submission and license information to be provided at a later time. Feel free to submit issues. Outside contributions will not be accepted before a license is posted.
